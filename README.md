# healthapp

Pet project for store health data

## Stack

-   Python (Django 4.0) + DRF (django-rest-framework) for backend and API
-   React.js (web-app)

## Objectives

App that provide user:

-   authenticate
-   load pdf with medical data
-   renders beuty charts etc
